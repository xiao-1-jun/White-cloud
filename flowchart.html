📊 流程步骤详解

🎯 阶段一：创建基础文件

  • 创建 Python 学习文件（learning_basics.py）
  • 创建个人网页（index.html）

⚙️ 阶段二：配置 Git 环境

  • 配置 Git 用户名和邮箱
  • 初始化本地仓库（git init）
  • 添加并提交文件（git add → git commit）

🏠 阶段三：创建 GitHub 仓库

  • 第一次创建仓库名 "-" 失败（URL 解析问题）
  • 修改仓库名为 "White-cloud"

🔑 阶段四：生成 GitHub 令牌

  • Fine-grained Token 权限不足
  • Classic Token 权限不完整
  • 最终生成完整权限的 Classic Token

🚀 阶段五：推送代码

  • 配置远程仓库地址
  • 拉取远程内容（git pull）
  • 提交并推送（git push）
  • 安全扫描拦截，移除令牌后重新推送

🌐 阶段六：部署网站

  • 启用 GitHub Pages
  • 配置 main 分支和 /root 目录
  • 网站上线：https://xiao-1-jun.github.io/White-cloud/
