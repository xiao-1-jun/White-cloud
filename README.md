# 🎬 类抖音网页端刷视频功能（竖向全屏版）

## 📖 项目简介

本项目实现了网页端模拟抖音的核心刷视频体验，支持竖向全屏视频流、鼠标滚轮切换、自动静音播放、常驻播放控件等功能，前期使用本地静态视频，后续可快速扩展线上视频接口对接。

## ✨ 核心功能

- ✅ 竖向全屏刷视频：单页一视频，沉浸式体验
- ✅ 鼠标滚轮切换：上下滚动自动切换视频，无额外过渡动画
- ✅ 自动播放控制：进入视口自动静音播放，离开视口自动暂停
- ✅ 常驻播放控件：进度条、时间显示、倍速、音量、全屏功能常驻显示
- ✅ 循环播放可选：用户可自主开启/关闭视频循环
- ✅ 半透明顶部导航：默认高透明度，不干扰视频观看
- ✅ 可扩展架构：预留线上视频接口对接能力，支持后续迭代

## 🆕 新增用户基础功能

### 1. 视频上传
- 支持本地视频选取上传，显示上传进度
- 限制视频格式与大小，自动适配竖屏播放比例
- 上传成功自动加入视频流，可直接滑动浏览
- 预留云端存储接口，支持线上批量存储

### 2. 用户关注系统
- 视频页展示创作者信息，一键关注/取关
- 独立关注列表、粉丝列表页面
- 支持查看关注用户专属视频内容

### 3. 个人资料管理
- 自定义更换用户头像
- 自由修改昵称、个人简介
- 展示个人作品数量、粉丝、关注数据
- 所有资料修改实时保存生效

## 🛠️ 技术栈

- **前端框架**: Vue 3 + Vite
- **路由管理**: Vue Router 4
- **状态管理**: localStorage 本地缓存
- **样式**: CSS3 (Flexbox、Grid)
- **播放控制**: IntersectionObserver + HTML5 Video API

## 🔧 功能说明

### 1. 滚轮切换视频
- 监听鼠标滚轮事件，每次滚动切换一个视频
- 禁用默认页面滚动，保证沉浸式刷视频体验

### 2. 自动播放 / 暂停
- 使用 IntersectionObserver 监听视频视口状态
- 视频进入视口时自动静音播放，离开视口时自动暂停并重置进度

### 3. 播放控件
- 常驻显示进度条、当前时间 / 总时长
- 支持 0.5x / 1x / 1.5x / 2x 倍速播放
- 音量控制、全屏切换、循环播放开关

### 4. 顶部导航
- 默认透明度 20%，不遮挡视频
- 鼠标悬停或点击时透明度变为 100%，支持页面跳转

### 5. 视频上传
- 点击或拖拽上传视频文件
- 实时预览、上传进度显示
- 自动保存到本地缓存

### 6. 用户关注
- 视频页一键关注 / 取消关注
- 关注列表、粉丝列表管理
- 实时状态切换

### 7. 个人资料
- 头像本地上传更换
- 昵称、个性签名编辑
- 作品、粉丝、关注数据统计

## 📝 发布优化要点

- 上传功能线上需开启浏览器本地文件读取权限，无需额外配置
- 用户资料全部使用浏览器本地缓存存储，发布后刷新页面数据不丢失
- 所有新增页面路由统一规划，保持网页端访问流畅
- 整体项目打包后体积轻便，发布加载速度快

## ⚠️ 注意事项

- **浏览器自动播放限制**: Chrome 等浏览器默认禁止非用户交互触发的有声播放，因此项目默认静音播放，符合抖音体验。
- **视频性能优化**: 项目默认仅加载当前视口的视频，其他视频暂停，避免资源占用过高。
- **跨域问题**: 后续对接线上视频资源时，需确保服务器配置 CORS 允许跨域访问。
- **兼容性**: 支持所有现代浏览器（Chrome、Firefox、Edge、Safari 11+），不支持 IE 浏览器。
